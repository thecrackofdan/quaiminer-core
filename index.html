How It Works

Install Drivers

Install AMD or NVIDIA GPU drivers (one-time setup). For AMD RX 590, we'll help you with automated setup.

2

Install QuaiMiner OS

Run the installer. It automatically installs quai-gpu-miner, sets up the service, and configures everything.

3

Configure & Start

Point it at your node's stratum proxy address. Mining starts automatically and runs on boot.

Quick Install

Clone the repository and run the installer:

git clone https://github.com/thecrackofdan/quaiminer-core.git
cd quaiminer-core/quaiminer-os
sudo ./install.sh

Configure Your Node

After installation, configure your node's stratum proxy:

sudo quaiminer-config set-stratum stratum://localhost:3333
sudo quaiminer-config set-node-rpc http://localhost:8545
sudo systemctl start quaiminer
