System Requirements

Hardware

  • AMD or NVIDIA GPU
  • 4GB RAM minimum
  • 10GB free disk space
  • USB 2.0+ port (for bootable)
  • Network connection

Software

  • Linux kernel 5.4+
  • AMD: ROCm 5.0+ or Mesa
  • NVIDIA: Driver 470+
  • Quai Network node (for solo mining)

Network

  • Internet connection
  • Port 3000 (dashboard)
  • Port 8545 (Quai node RPC)
  • Port 3333 (stratum proxy)
